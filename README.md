# Study Time Tracker
This Python program estimates your weekly study hours based on how many hours you studied today. It was created for the Python Project + GitHub Workflow assignment for the Muma College of Business at the University of South Florida.

## How to Run
Run the program using:
python app.py
or
python3 app.py

## Example Output (Studying 12 Hours per Week)
How many hours did you study today? abc
Please enter a valid number.
